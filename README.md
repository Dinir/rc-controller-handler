## 동일한 작업환경을 구축하는 법
1. Unity 6000.2.6f2[^버전-체인지]를 설치합니다. 모듈은 Windows 하나로 일단은 충분합니다.  
   해당 버전은 Unity Hub에서 Installs -> Install Editor -> Archive로 가서 찾으셔야 합니다.
3. 빈 프로젝트를 만드시고,
4. Assets 폴더를 가져와서 거기에 덮어씌우세요.
